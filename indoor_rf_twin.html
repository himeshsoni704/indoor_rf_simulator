RF heatmap: FSPL = 32.44 + 20·log₁₀(d_km) + 20·log₁₀(f_MHz) minus wall losses. Recomputed only when routers or network change.

RTI (Wilson & Patwari 2010): Agent positions synthesise simulated link RSSI readings — what real hardware would measure. The reconstruction algorithm itself never reads agent coordinates; it only sees numeric signal drops per link and back-projects via w = exp(−(dA+dB−|AB|)/λ), normalised by link density. Red dashes on RTI panel = WiFi mesh links below −85 dBm.

Placement score: 60 % coverage + 20 % avg link length + 20 % angular diversity. Angular diversity is the circular std-dev of link bearings — penalises all-routers-on-one-wall layouts that produce parallel (correlated) links.
